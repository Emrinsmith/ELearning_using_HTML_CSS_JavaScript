Lets Divide Data Science into 4 parts

Statics
Static data structures are characterized by their fixed size and predefined memory allocation.
Machine Learning
Data science and machine learning are related fields that both use data to gain insights, but they have different focuses.
Programming
Programming in data science is the use of coding languages to perform tasks such as: Extracting data from multiple datasets, Cleaning and analyzing data, Visualizing data, Designing databases, and Building machine learning algorithms.

Lets Divide Devops into 5 parts

DevOps collaboration tools
These tools help DevOps teams communicate, organize work, and share knowledge. They can also send notifications from automated systems to team members.
Continuous deployment
This is the phase in the DevOps life cycle where code is deployed to production servers. DevOps tools that can support frequent and fast deployments are essential for this phase.
DevSecOps
This is a type of DevOps that integrates security practices into the development and operation processes.
Ansible
Ansible is an open-source tool that helps with integrating internally developed applications into programs. It can also be used to automate, configure, orchestrate, and manage IT infrastructure.
Monitoring
This is a continuous process that provides real-time information about the entire development pipeline. A monitoring solution can aggregate and analyze data from various sources to identify patterns and generate alerts.
